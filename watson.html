What Watson Does
The Engine Room

Autonomous Investigation

Type a target — person, company, domain, wallet, or email. Watson classifies it, plans the investigation, dispatches 4–6 parallel angles across 16 APIs, and streams findings in real time.

Persistent Memory Graph

Every investigation writes to a knowledge graph. Entity #129 from Case #7 connects to Entity #341 from Case #42. Cross-case intelligence surfaces automatically — no agent does this.

Source-Verified Intelligence

Every finding carries a source URL, confidence tier (CONFIRMED / PROBABLE / POSSIBLE / UNVERIFIABLE), and provenance. When data can't be verified, the report says so explicitly.

Community Graph (MCP)

Per-case consent publishing to a shared community knowledge graph via MCP protocol. Other instances query prior intelligence before investigating. 5 MCP tools available.

Provider-Agnostic LLM

Works with DeepSeek, OpenAI, Anthropic, Groq, or any OpenAI-compatible endpoint. You own your model choice. ~$1/mo in LLM costs at typical usage.

Runs Locally. Your Data, Your Machine.

Self-hosted. Reports save as markdown on your disk. Entity graph persists in JSONL. No cloud dependency, no telemetry. Air-gapped investigations possible.

Get Started in 30 Seconds
Quick Start
git clone https://github.com/Lorenzobaron99/watson-osint.git
cd watson-osint
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# Web UI (recommended)
PYTHONPATH=.:src uvicorn watson.web.app:app --port 8777

# CLI
python -m watson.cli

Open http://localhost:8777 → type a target → watch the 7-phase pipeline run live. Reports save as CASE-XXXX.md in ~/watson-cases/.
