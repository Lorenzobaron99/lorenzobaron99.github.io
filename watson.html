What Watson Does

🔍

Autonomous Investigation

Type a target — person, company, domain, wallet, or email. Watson classifies it, plans the investigation, dispatches 4–6 parallel research angles across 16 APIs, and streams findings in real time. No manual query-hopping between tools.

🧠

Persistent Memory Graph

Every investigation writes to a knowledge graph. Entity #129 from Case #7 connects to Entity #341 from Case #42. Watson surfaces cross-case connections automatically — no agent does this.

🔗

Source-Verified Intelligence

Every finding carries a source URL, confidence tier (CONFIRMED / PROBABLE / POSSIBLE / UNVERIFIABLE), and provenance. When data can't be verified, the report says so explicitly. No hallucinated intelligence.

🌐

Community Graph (MCP)

Per-case consent publishing to a shared community knowledge graph. Other Watson instances can query prior intelligence on a target before investigating. Protocol-native MCP server with 5 tools.

⚡

Provider-Agnostic LLM

Works with DeepSeek, OpenAI, Anthropic, Groq, or any OpenAI-compatible endpoint. You own your model choice. No vendor lock-in. ~$1/mo in LLM costs at typical usage.

🏠

Runs Locally. Your Data, Your Machine.

Self-hosted. Reports save as markdown on your disk. Entity graph persists in JSONL. No cloud dependency, no telemetry, no external data exfiltration. Air-gapped investigations are possible.

Who This Is For

OSINT practitioners who run the same searches across 10+ tools manually. Watson parallelizes the boring part and gives you a report with sources you can verify.
Investigators & analysts who need cross-case intelligence — you investigated Company A in March, now Company B surfaces the same director. Watson connects the dots.
Journalists & researchers doing due diligence, source verification, corporate research. Watson gives you structured briefings with provenance instead of raw search results.
Security teams running supply chain risk, executive protection, threat intelligence. Watson's graph builds an institutional memory that doesn't leave with the analyst.

Not for people who want a ChatGPT wrapper with investigative cosplay. Watson calls real APIs, not LLM hallucinations dressed as intelligence.

Quick Start

git clone https://github.com/Lorenzobaron99/watson-osint.git
cd watson-osint
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# Web UI (recommended)
PYTHONPATH=.:src uvicorn watson.web.app:app --port 8777

# CLI
python -m watson.cli

Open http://localhost:8777 → type a target → watch the 7-phase pipeline run live. Reports save as CASE-XXXX.md in ~/watson-cases/. Entity graph persists in ~/.watson/graph/.
